# Chaturbate DVR 项目构建完成

## 项目概述
基于 Spring Boot + MyBatis + MySQL + Vue3 + Element Plus 的直播录制管理系统。

## 已分析的关键信息

### API接口结构
- **URL**: `https://zh-hans.chaturbate.com/api/chatvideocontext/{username}/`
- **关键字段**:
  - `room_status`: public/private/offline - 判断是否录制
  - `hls_source`: HLS主播放列表地址 - 录制源
  - `broadcaster_username`: 主播用户名
  - `num_viewers`: 观看人数
  - `room_title`: 房间标题

### HLS流格式
- 主播放列表(Master Playlist)包含多个质量的流
- 可用质量: 360p(640x360), 480p(852x480), 540p(960x540), 720p(1280x720), 1080p(1920x1080)
- 每个流对应一个chunklist，包含TS片段
- 音频和视频分离，通过AUDIO属性关联

### 反爬机制
- 需要 `cf_clearance` Cookie
- 需要正确的 `User-Agent`
- 需要 `Referer` 头

## 核心功能实现

1. **API调用**: 使用Apache HttpClient 5，携带Cookie和User-Agent访问直播间接口
2. **HLS解析**: 自定义解析器处理m3u8主播放列表和片段列表
3. **录制逻辑**: 
   - 解析主播放列表选择质量
   - 循环下载chunklist获取片段URL
   - 下载TS片段追加到文件
   - 每5秒刷新chunklist
4. **自动监控**: Spring Scheduled任务每30秒检查直播间状态
5. **Web界面**: Vue3 + Element Plus管理界面
